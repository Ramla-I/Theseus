Expand description

Lock all shared states (i.e. those defined in lazy_static!) and execute the closure. This function is mainly provided for the shell to clean up the pointers stored in the map without causing a deadlock. In other words, by using this function, shell can prevent the application from holding the lock of these shared maps before killing it. Otherwise, the lock will never get a chance to be released. Since we currently don’t have stack unwinding.
